# Web Scraper (Python)

This project extracts data from a website using Python, requests, and BeautifulSoup.

## Features
- Sends HTTP requests to websites
- Parses HTML content
- Extracts specific data (titles, text, etc.)
- Displays results in the terminal

## Technologies
- Python
- requests
- BeautifulSoup (bs4)

## Installation

```bash
pip install requests beautifulsoup4
